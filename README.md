# rabbitmq-reusable-class


## USAGE

- First step
  ```
  docker run -it --rm --name rabbitmq -p 5672:5672 -p 15672:15672 rabbitmq:3.12-management
  ```

- Second step
  
  Run this command in first terminal.

  ```
  npm run publisher
  ```
  
  Run this command in second terminal.
  ```
  npm run subscriber
  ```
